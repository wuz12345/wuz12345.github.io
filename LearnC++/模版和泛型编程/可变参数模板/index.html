    可变类型模板

    可变参数模板(Variadic Templates)¶

    C++11后模板支持可变参数，允许在使用类模板时传入任意个数的参数。

    可变参数模板(Variadic Templates)¶

    下面函数中print函数支持多个参数。 

    #include <iostream>

// 这个空函数用于退出递归
void print ()
{
}

template<typename T, typename... Types>
void print (T firstArg, Types... args)
{
  std::cout << firstArg << '\n';  // print first argument
  print(args...);                 // call print() for remaining arguments
}

    上面的函数模板会递归的打印所有函数参数。除第一个参数外，其余参数通过args标识:

    void print (T firstArg, Types... args)

    使用过程如下:

    std::string s("world");
print (7.5, "hello", s);

// 第一个参数:
print<double, char const*, std::string> (7.5, "hello", s);
// 第二个参数:
print<char const*, std::string> ("hello", s);
// 第三个参数
print<std::string> (s);
// 空参数
print() //空函数

    重载可变参数和非可变参数模板(Overloading Variadic and Nonvariadic Templates)¶

    下面这个代码和上一节的代码等价。 

    #include <iostream>

template<typename T>
void print (T arg)
{
  std::cout << arg << '\n';  // print passed argument
}

template<typename T, typename... Types>
void print (T firstArg, Types... args)
{
  print(firstArg);           // call print() for the first argument
  print(args...);            // call print() for remaining arguments
}

    这段代码，说明在函数重载时优先选择无可变参数的版本。

    sizeof...操作符(Operator sizeof…)¶

    sizeof...操作符获取参数个数。

    template<typename T, typename... Types>
void print(T firstArg, Types... args) {
    std::cout << sizeof...(Types) << '\n';
    std::cout << sizeof...(args) << '\n';
}

    需要注意的是，下面的用法是错误的:

    template<typename T, typename... Types>
void print(T firstArg, Types... args) {
    std::cout << firstArg << '\n';
    if (sizeof...(args) > 0) {
        print(args...);
    }
}

    因为函数会实例化所有的if分支，而上面的代码没有sizeof...(args) == 0的实现。

    折叠表达式(Fold Expressions)¶

    从C++17开始，提供了使用二元运算符操作可变参数的特征:

    template<typename... T>
auto foldSum (T... s) {
    return (... + s); // ((s1 + s2) + s3) ... 
}
    除&&,||和,外，其他运算符不能处理sizeof...(s)==0的情况。

    折叠表达式的规则如下:

    1. (... op pack)—> (((pack1 op pack2) op pack3)... op packN)
    2. ( pack op ... )—> (pack1 op (... (packN-1 op packN)))
    3. ( init op ... op pack )—> (((init op pack1) op pack2)... op packN)
    4. ( pack op ... op init )—> (pack1 op (... (packN op init)))

    可以使用->*来遍历二叉树:

    // define binary tree structure and traverse helpers:
struct Node {
  int value;
  Node* left;
  Node* right;
  Node(int i=0) : value(i), left(nullptr), right(nullptr) {
  }
  //...
};
auto left = &Node::left;
auto right = &Node::right;

// traverse tree, using fold expression:
template<typename T, typename... TP>
Node* traverse (T np, TP... paths) {
  return (np ->* ... ->* paths);      // np ->* paths1 ->* paths2 ...
}

int main()
{
  // init binary tree structure:
  Node* root = new Node{0};
  root->left = new Node{1};
  root->left->right = new Node{2};
  //...
  // traverse binary tree:
  Node* node = traverse(root, left, right);
  //...
}

    路径是:

    (np ->* ... ->* paths)

    可变参数模板的应用(Application of Variadic Templates)¶

    可变参数模板的应用在STL中有很多应用，比如模板类中的构造对象:

    // create shared pointer to complex<float> initialized by 4.2 and 7.7:
auto sp = std::make_shared<std::complex<float>>(4.2, 7.7);
// call foo(42,"hello") in a separate thread
std::thread t (foo, 42, "hello");
// insert a Customer initialized by three arguments
std::vector<Customer> v;
...
v.emplace("Tim", "Jovi", 1962);

    在上面的例子中，涉及到了转发和移动语义，即:可变参数是转发forward给构造函数的。例子中涉及到的std::make_shared<>,std::thread,std::vector的声明如下:

    namespace std {

template<typename T, typename... Args> shared_ptr<T>
make_shared(Args&&... args);

class thread {
public:
    template<typename F, typename... Args> 
    explicit thread(F&& f, Args&&... args); 
    ...
};

template<typename T, typename Allocator = allocator<T>>
class vector {
    public:
    template<typename... Args> reference emplace_back(Args&&... args); 
    ...
}; 
}

    可变参数模板在函数模板中也有类似的应用，传值类型衰变，传引用不会。

    // args are copies with decayed types:
template<typename... Args> void foo (Args... args);
// args are nondecayed references to passed objects: 
template<typename... Args> void bar (Args const&... args);
